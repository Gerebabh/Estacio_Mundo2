Listas em HTML

<ol> - Ordered list (Lista Ordenada)
A tag <ol> é usada para criar uma lista ordenada, onde cada item é numerado em sequência. É comumente usada para criar listas de passos, etapas ou qualquer coisa que precise ser apresentada em ordem específica.

<ul> - Unorder list (Lista Não Ordenada)
É usada para criar uma lista não ordenada, onde cada item é precedido por um marcador, geralmente um ponto preto ou outro símbolo. É usada para criar listas de itens que não têm uma ordem específica.

<li> - List Item (Item da lista)
Essa tag é usada para criar um item em uma lista, seja uma lista ordenada (<ol>) ou uma lista não ordenada (<ul>). Cada <<li>> representa um elemento individual na lista e é usado para marcar um item específico na sequência da lista.

<dl> - Definition list (Lista de definição)
É usada para criar uma lista de definição, onde você pode definir termos e suas respectivas definições. Ela é frequentemente usada para criar glossários ou para apresentar informações em que você deseja destacar os termos e explicar seus significados.

<dt> - Definition term (Termo de definição)
É usada dentro de uma lista de definição (<dl>) para representar um termo ou item que você deseja definir. Geralmente, os termos são exibidos em negrito ou de alguma maneira para destacá-los dos detalhes de definição.

<dd> - Definition Deatils (Detalhes de definição)
A tag <dd> é usada dentro de uma lista de definição (<dl>) para fornecer os detalhes, descrições ou definições associadas aos termos definidos pela tag <dt>. Ela fornece informações mais elaboradas sobre os termos da lista de definição.
